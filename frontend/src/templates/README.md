# Templates

Alle Templates sind hier gebündelt:
- `layout/` enthält die aktiven Layout-Komponenten, die im Projekt gerendert werden.
- `library/` enthält reine Copy-&-Paste Snippets mit Überschriften (z. B. "Large Base Button").

Wenn du einen Baustein kopieren möchtest, öffne die passende Datei in `library/` und suche nach der Überschrift.
